# Starter Proj
## Steps to setup and run
1. `git clone git@github.com:dani7beth/daily-horoscope.git` CHANGE THIS TO YOUR REPO
2. `cd project-name`
3. in database.yml change name of database `what_ever_you_named_your_start_project` to **project-name**
4. `yarn`
5. `yarn start`
